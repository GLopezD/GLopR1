How to use the GLopR1 package

Introduction

This package is intended to provide statistics graphs, such as histogram and boxplot for a set of data. In addition, this tool can ckeck whether or not the data come from a normally distributed population. Finally, a dataframe with a summary of the descriptive statistics of the data.

Example

library(GLopR1)
x=rnorm(100, 1,2)
f1(x)
